# Data Subject Rights (DSR) Portal

## 📌 Project Overview
The Data Subject Rights (DSR) Portal is a web application that allows users to submit requests related to their personal data and enables administrators to manage and process those requests efficiently.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Submit Data Requests
- View Dashboard
- Logout functionality

### 🛠 Admin Module
- View all user requests
- Approve / Reject requests
- Search and filter requests
- Admin dashboard interface

---

## 🔐 Security
- Basic authentication using localStorage
- Role-based access (Admin/User)
- Restricted access to admin and dashboard pages

---

## 🧰 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Spring Boot (Java)
- Database: H2 / MySQL
- Tools: Postman, GitHub
